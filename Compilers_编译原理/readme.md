## 关于考试要点
- 主要是课后习题(特别是布置过的),无论如何都要做一遍过去
- 主要内容在前半学期,
  - 文法推导
  - 语法分析(一般都会考)
    - LL1分析
    - LR(0)&LR(1)分析(二选一考察)
- 后半学期的内容要求的比较简单(尽管不代表知识简单)
    - 活动记录
    - 基本快/流图
    - 语法制导翻译(简单考察)
- 选择题方面涉及较广,但主要是概念,课本&课件多看看,或者网络习题搜搜看
